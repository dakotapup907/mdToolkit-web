Update

The [mdToolkit project](https://www.mdtoolkit.org) does not currently have a public instance of a Rails server. We appologize for the inconvienence. The mdEditor and the other components of the mdToolkit have seen increased use. We are a small project and do not currently have the resources necessary to host a generally accessible server. An instance of the mdTranslator should be available at:

[https://api.sciencebase.gov/mdTranslator/api/v3/translator](https://api.sciencebase.gov/mdTranslator/api/v3/translator)

If necessary, the `mdTranslator API URL` can be manually configured in the mdEditor under the `Sttings` menu. The mdTranslator API URL should take the following form: [https://api.sciencebase.gov/mdTranslator/api/v3/translator](https://api.sciencebase.gov/mdTranslator/api/v3/translator)
